## ‚öôÔ∏è Core Features

### 1. Panic Index (Urgency Engine)
A weighted prioritization system that evaluates:
> **Difficulty √ó Time Remaining √ó Importance**

This generates a dynamic urgency score that helps users focus on what actually matters. Instead of guessing what to study, the app tells you where the highest ROI lies.

### 2. Gamified Progression System
Exam Quest transforms preparation into a progression loop:
* Scholar Levels
* XP accumulation
* Achievements
* Rank evolution

This shifts motivation from fear to identity. You‚Äôre no longer ‚Äústudying for exams.‚Äù You‚Äôre leveling up.

### 3. Quest-Based Planning
Subjects are converted into quests. Each quest includes:
* Difficulty selection
* Session timing (FN / AN)
* Countdown to execution

This creates a sense of momentum and intentional planning.

### 4. Behavioral Feedback Loops
To reinforce consistency, the system includes:
* Streak tracking
* Progress insights
* Real-time motivational cues
* Reward popups (Aura system)

These loops are designed to encourage repeat engagement and habit formation.

### 5. System Insights Layer
The app provides contextual nudges like: *"DSA is your current focus. Keep the momentum."* This introduces lightweight intelligence without overwhelming the user.
